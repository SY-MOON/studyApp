# satudy-app
매주 토요일 개발 스터디 출석체크/지각비 계산용 앱

## 기능
1. 출석체크
2. 지각(또는 결석)시 시간에 따라 지각(결석)비 계산
3. 불참 또는 지각 예정자 미리 일정 등록
